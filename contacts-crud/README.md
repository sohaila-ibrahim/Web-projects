# Contacts CRUD

A simple Contacts Management application built with HTML, CSS, Bootstrap, and JavaScript.

## Features

- Add new contacts
- Update existing contacts
- Delete contacts
- Search contacts
- Store data using Local Storage
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Local Storage
