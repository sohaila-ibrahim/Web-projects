# 🍽️ What's For Dinner

A responsive recipe discovery web app that helps you decide what to cook. Click a button and get a randomly picked recipe complete with ingredients, step-by-step instructions, nutrition facts, and chef's tips.

**Live Demo:** https://sohaila-ibrahim.github.io/Web-projects/whats-for-dinner/

## Features

- Randomly displays a new recipe on each click (never repeats the same one twice in a row)
- Full ingredients list with numbered steps
- Step-by-step cooking instructions
- Nutrition breakdown (calories, protein, carbs, fat, fiber, sodium) with color-coded icons
- Chef's tips for each recipe
- Extended prep time warning for recipes over 45 minutes
- Fully responsive design using Bootstrap

## Built With

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla)
- Font Awesome
